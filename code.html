Code explanation below.

getValue()

This is the controller function that is responsible for receiving and then sending the input provided bu the user. It will pass the input through the logic function and then through the view function to return their input string reversed.

reverseString()

This function is the logic function and will break down the recieved string into substrings inside of an array. It then reverses the order of the substrings inside of the array. It then rejoins the now reversed substrings into a new string and passes it on to the display function

displayString()

The display function injects the new string into the html element and makes it appear on the display for the user.
